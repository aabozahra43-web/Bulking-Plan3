🍽️ نظام غذائي لزيادة الوزن والعضل

مخصص لشخص طوله ١٧٧ سم، وزنه ٦٣ كجم، عمره ١٩ سنة

السعرات: ١٧٠٠ - ١٩٠٠ سعرة/يوم البروتين: ١٠٠ - ١٢٠ جرام/يوم ٣ وجبات رئيسية

الوجبة ١: الفطار

اختيار أ
٤ بيضة كاملة + ٢ رغيف عيش بلدي + جبنة قريش أو فيتا
≈ ٦٥٠ سعرة ≈ ٤٠ جم بروتين
اختيار ب
فول (٢ كوب) + زيت زيتون + رغيف عيش بلدي + بيضتين كاملتين
≈ ٧٠٠ سعرة ≈ ٣٢ جم بروتين
اختيار ج
٣ بيضة كاملة + جبنة بيضة + رغيف عيش بلدي + موزة
≈ ٦٢٠ سعرة ≈ ٣٠ جم بروتين

الوجبة ٢: الغداء

اختيار أ
فراخ مشوية (١٥٠ جرام) + رز أو مكرونة (كوب) + خضار سوتيه
≈ ٦٥٠ سعرة ≈ ٤٥ جم بروتين
اختيار ب
لحمة مفرومة أو كبدة (١٥٠ جرام) + بطاطس مسلوقة أو مهروسة + سلطة
≈ ٧٠٠ سعرة ≈ ٤٠ جم بروتين
اختيار ج
تونة (علبة) + أرز + خضار + زيت زيتون
≈ ٦٠٠ سعرة ≈ ٣٨ جم بروتين

الوجبة ٣: العشاء

اختيار أ
جبنة قريش أو فيتا + رغيف عيش بلدي + خضار (خيار وطماطم) + زيتون
≈ ٣٨٠ سعرة ≈ ٢٢ جم بروتين
اختيار ب
زبادي كامل الدسم + توست أو عيش بلدي + جبنة بيضة + خيار
≈ ٤٠٠ سعرة ≈ ٢٤ جم بروتين
اختيار ج
بيضتين مسلوقة + جبنة قريش + رغيف عيش بلدي + سلطة خضار
≈ ٣٦٠ سعرة ≈ ٢٦ جم بروتين

📌 ملاحظات مهمة

  • بين الوجبات: فاكهة (موز، تين، عنب) ومكسرات (لوز، فول سوداني) كسناكس لزيادة السعرات لو حسيت إنك محتاج سعرات زيادة.
  • الزيوت: زيت زيتون على السلطات والخضار لزيادة السعرات بشكل صحي.
  • اللبن: ٣-٤ أكواب لبن كامل الدسم يومياً موزعة على اليوم.
  • الماء: ٢.٥-٣ لتر يومياً.
  • الأرقام المذكورة تقريبية، الهدف منها التوجيه العام لا الدقة المعملية.
